
# Installation

`git clone https://github.com/nourpups/beginner_complete.git`

`cd beginner_complete`

`composer install`

`php artisan key:generate`

## Set database settings

`DB_DATABASE=your_db`

`DB_USERNAME=your_username`

`DB_PASSWORD=your_password`

## Migrations & Seeders

`php artisan migrate`

`php artisan db:seed`
